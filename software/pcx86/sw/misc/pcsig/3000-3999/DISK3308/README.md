                      Last updated:  January, 1992


This file includes sample descriptions which you may use to describe
the Word Hunt package.

Disk Vendors and Distributors please refer to the VENDOR.DOC text
file.


----------------------------------------------------------------------
                         Program Information:
----------------------------------------------------------------------


File Name(s) for BBSs:
----------------------

If the programs and documentation are contained within a single
archive file, please use the following name:

              WordHunt.ZIP   Word Hunt ver 2.0                    


If the program is compressed using some other file compression
method, then please use the name "WordHunt" with the appropriate
extension (PAK, ARC, LZH, ZOO, etc.).


Keywords:
~~~~~~~~~
   puzzle, word, search, game, boss


Category:
~~~~~~~~~
   Games - Word Games                                    


Program Name, Category, Keywords:
---------------------------------

   Word Hunt, game, puzzle, words, search, game, boss                 


Short Description:
------------------

   A word Search-and-Find type puzzle program. Several skill levels
   with printing, "Boss" key, and the option to enter your own words.


Long Description:
-----------------

   A word Search-and-Find type puzzle program. The program can create
   an extensive number of puzzles at different skill levels from it's
   stored word database. The skill levels cover puzzle blocks of 
   10 by 10 to up to 20 by 20. The skill level also determines the
   number of directions the words can be placed in the block. There
   are options for saving and restoring a puzzle from a file, changing
   the screen colors, using a mouse, and printing out the puzzle.  In 
   addition, the user can enter his own selection of words from which
   to create a puzzle.  A "Boss" key is available to bring up a fake 
   DOS screen (for those who like to play at work).



Registration Information:
-------------------------

   All registered users will recieve confirmation of their registration,
   and three months of technical support (mail or through Compuserve).

   For complete user registration information please refer to the
   README.DOC text file.
