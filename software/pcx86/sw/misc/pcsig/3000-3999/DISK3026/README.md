## README.TXT

{% raw %}
```
                                 Whoop It Up! (tm)
                                 For Windows


    What is Whoop it Up!?

    Whoop It Up! is a program designed to maximize the sound functions
    within Windows.  With Whoop It Up! you can add sound not only to many
    system events, but to individual applications as well.  Whoop It Up!
    supports any MIDI or WAVE device supported by Windows 3.1. You can also
    add sound to Message Box icons, even in programs that don't support
    Windows 3.1 sound events.

    To install the evaluation version of Whoop It Up!, just copy all the
    included files into a subdirectory.  From File Manager, double click
    on WHOOP.EXE to start the program.  You can drag WHOOP.EXE into your
    Program Manager Startup group and Whoop It Up! will start automatically
    every time you start Windows.

    If you use this program we would like to request that you pay  for  it!
    If you send in your hard earned money, count on us to ship out $4.95 to
    the  World  Wildlife  Fund,  an  organization  that  helps  protect our
    endangered wildlife around the world. Please take the time to pull down
    the about box for more information.

    If you plan on using this program with your PC's speaker you will need
    to obtain a copy of the PC Speaker Driver from Microsoft.  GO MSL at
    the ! prompt in Compuserve and download SPEAK.EXE, a self-extracting
    ZIP file with the PC Speaker Driver and installation instructiosn. The
    PC Speaker Driver is also available without charge on the Microsoft
    Download Service BBS (206) 936-MSDL.

    If you have further questions regarding the installation of this
    software or require assistance please call 1-900-88-4HELP. There is a
    small charge for this service. When you register this software through
    our distributor you will receive the latest version of "Whoop it Up" on
    either a 1.44meg 3 1/2 or 1.2meg 5 1/4 and operation booklet and
    technical support, plus over a megabyte of WAV & MIDI files.

    Additional WAV and MIDI files  are  being  developed  and  will  become
    available soon!

    Windows is a registered trademark of Microsoft Corp.
    Whoop It Up! is a trademark of Starlite Software Company.
```
