Kitty's Computer Game
(c)1992 T42 Software
Programmed by Peter de Vocht
Graphics and design by Kevin Phillips

Kitty's Computer Game is a rather primitive maze game for young
children between the ages of 3 & 5 years. The aim of the game is to
complete 3 tasks for Kitty. Each task involves collecting 3 different
objects from 3 mazes. First, Kitty will tell you the task he wishes you
to perform (ie. He is either Hungry, Playful or Tired), then you will be
shown the object to collect, and you will have to move Kitty around
the maze to the object using the arrow keys (No mouse is supported)
The aim is simply to complete each level without collecting the wrong
objects. ESC aborts the current game.

The game features 256 colour fully animated graphics (320x200), 
inbetween-level animated screens and Soundblaster/Ad-lib support
for some nice music.

Minimum Requirements are :
80286 (12 Mhz) or better,
VGA Graphics card + monitor,
640 K Ram.

Note : On slower machines (ie. 8 or 10 Mhz) some of the graphics will
look a little jerky ! Also, faster machines (ie. 50 Mhz) will be too fast for 
some of the inbetween-level animations ! Please also note, we haven't had
the chance to test this game with an Adlib card. The music was written 
with a soundblaster card, but hopefully it should work fine with an Adlib.

Controls for the game are :	Cursor keys - Move
				ESC to abort while in game
