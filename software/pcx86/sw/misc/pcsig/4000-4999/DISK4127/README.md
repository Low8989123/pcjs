Softline World Clock includes InWin: an installer
           program run from within Windows.

NOTE: This program should be installed to the hard
      drive. Allow about 2 minutes for the install
      process.

Step #1: Make a copy of the Master diskette and keep the
         master diskette in a safe place.
Step #2: Insert the copy in Drive A or B.
Step #3: In Windows Program Manager, select File-Run...
Step #4: Type:   A:INSTALL  or    B:INSTALL
         Follow the install instructions as they appear
         on the screen.
Step #5: After the Install is complete, double click
         the left mouse button on the World Clock icon
         to start the program.
Step #6: Select OK at the License reminder screen.
Step #7: For instructions (HELP), press the <F1> key and/or
         click on the icon titled "Manual" in the Softline
         World Clock Program Group.
Step #8: Please send in your license fee.  Softline will 
         remove the pester screen, send a manual, provide
         extended font capability in the scrolling Time
         Bar (Windows 3.1 or greater users), provide 
         additional information in Help <F1> file on
         various countries, and the most recent version
         of the software.  Thank you. 
	   Select the Write icon titled "Registration"
           for the registration/order form and information
           on other Softline products.

        for teaching children about Time Zone relationships.

        reference to the local time in nine different cities
        around the world and are updated to the minute.  Double
        click on the map for the current time in any of the 24
        Time Zones.  The Auto-Stretch/Shrink feature sizes the
        World Clock map to fit anywhere onscreen.  "Always on
        Top" command to keep the World Clock window in the
        foreground.  On-Line Help <F1> for countries on Daylight
        Savings Time, Odd-Hour, and Special Exceptions as well
        as command explanation/selection.  Includes InWin: 
        Windows utility program for an easy install within
        Windows.  Substitute your own colorful map of the world
        for the Softline map (Requires .BMP format and some
        adjustment for Time Zone alignment).  Softline World Clock;
        a great addition to your library of Windows utilities!.
